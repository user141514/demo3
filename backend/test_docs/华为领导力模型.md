# 华为领导力模型

## 概述

华为领导力模型基于"以客户为中心、以奋斗者为本、长期艰苦奋斗、坚持自我批判"的核心价值观，构建了完整的领导力发展体系。

## 核心维度

### 1. 战略思维力
领导者必须具备前瞻性战略视野，能够洞察行业趋势，识别技术变革信号，制定3-5年战略规划。

**行为标准：**
- 每年主导一次战略规划研讨会，输出清晰的战略地图
- 季度性开展行业趋势分析，形成竞争态势报告
- 将公司战略目标拆解为可执行的部门级行动计划

### 2. 组织建设力
打造高绩效团队是华为领导者的核心职责。包括人才选拔、梯队建设、绩效管理和文化传承。

**行为标准：**
- 每半年完成一次人才盘点，建立关键岗位继任计划
- 主导跨部门人才轮岗项目，培养复合型管理人才
- 建立团队内部导师制，确保知识和经验的代际传承

### 3. 客户导向力
深刻理解客户需求，将客户声音转化为内部改进行动。从被动响应客户投诉转变为主动预测客户需求。

**行为标准：**
- 每季度至少拜访3个关键客户，输出客户需求洞察报告
- 组织跨部门客户痛点分析会，推动端到端流程优化
- 建立客户满意度闭环管理机制

### 4. 创新驱动力
在技术快速迭代的时代，领导者需要持续推动技术创新和商业模式创新。

**行为标准：**
- 每年设立创新基金，支持和孵化至少2个创新项目
- 建立创新容错机制，鼓励试错和快速验证
- 定期组织技术前沿分享会，保持团队对新技术的敏感度

### 5. 执行突破力
将战略构想转化为实际成果。面对困难和阻力时，具备突破性思维和攻坚克难的决心。

**行为标准：**
- 对重大项目实行"作战室"管理模式，集中资源打歼灭战
- 建立关键里程碑跟踪机制，每周检视关键节点推进情况
- 在遇到重大障碍时主动向上求助和横向协调，不等待不拖延

### 6. 全球视野力
适应国际化业务需求，具备跨文化管理能力和全球资源整合能力。

**行为标准：**
- 深入了解目标市场的法律法规和商业文化
- 建立全球化人才配置机制，实现关键岗位的国际化轮岗
- 主持跨国项目时充分考虑文化差异，采用包容性决策方式

## 层级差异

### 高层（总裁/副总裁）
- 侧重：战略方向制定、组织能力建设、关键干部任用
- 产出：行业趋势洞察报告、3-5年战略规划、组织变革方案

### 中层（总监/总经理）
- 侧重：部门战略承接、跨部门协同、重点项目突破
- 产出：年度经营计划、协同项目管理表、团队能力评估报告

### 基层（主管/组长）
- 侧重：任务执行、团队带领、一线问题解决
- 产出：周/月度工作计划与复盘、团队成员绩效反馈、现场改善提案
